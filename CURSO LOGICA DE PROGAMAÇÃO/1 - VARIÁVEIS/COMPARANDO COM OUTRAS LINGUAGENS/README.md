# Comparando Variáveis Com Outras Linguagens

Em JavaScript, uma variável pode ser criada com `let`, `const` ou `var`. Em outras linguagens, a ideia é a mesma: guardar um valor em um nome para usar depois.

## Java

```java
String notificacao = "Aplicativo diz:";
int idadeAluno = 20;
boolean selecionavel = true;
```

Java exige informar o tipo da variável, como `String`, `int` e `boolean`.

## C Sharp

```csharp
string notificacao = "Aplicativo diz:";
int idadeAluno = 20;
bool selecionavel = true;
```

C# também usa tipos definidos, mas os nomes são parecidos com JavaScript em alguns casos.

## Python

```python
notificacao = "Aplicativo diz:"
idade_aluno = 20
selecionavel = True
```

Python não usa `let` nem precisa declarar o tipo antes do nome da variável.

## PHP

```php
$notificacao = "Aplicativo diz:";
$idadeAluno = 20;
$selecionavel = true;
```

PHP usa `$` antes do nome da variável.
