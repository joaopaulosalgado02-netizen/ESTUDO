# Comparando Operadores Aritméticos Com Outras Linguagens

Operadores aritméticos fazem contas: soma, subtração, multiplicação, divisão e resto da divisão.

## Java

```java
int preco = 10;
int quantidade = 3;
int total = preco * quantidade;
int resto = 10 % 3;
```

Java usa os mesmos símbolos principais de JavaScript: `+`, `-`, `*`, `/` e `%`.

## C Sharp

```csharp
int preco = 10;
int quantidade = 3;
int total = preco * quantidade;
int resto = 10 % 3;
```

C# também usa os mesmos operadores matemáticos.

## Python

```python
preco = 10
quantidade = 3
total = preco * quantidade
resto = 10 % 3
```

Python usa os mesmos símbolos, mas não precisa de ponto e vírgula.

## PHP

```php
$preco = 10;
$quantidade = 3;
$total = $preco * $quantidade;
$resto = 10 % 3;
```

PHP usa os mesmos operadores, mas as variáveis começam com `$`.
