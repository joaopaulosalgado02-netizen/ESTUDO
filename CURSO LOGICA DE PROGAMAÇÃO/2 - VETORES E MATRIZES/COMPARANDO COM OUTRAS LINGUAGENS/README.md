# Comparando Vetores E Matrizes Com Outras Linguagens

Em JavaScript, arrays guardam listas de valores. Matrizes são arrays dentro de arrays.

## Java

```java
String[] nomes = {"João", "Maria", "Carlos"};
String[][] pessoas = {
    {"João", "M", "20 anos"},
    {"Maria", "F", "25 anos"}
};
```

Java usa `[]` para indicar listas e exige que todos os itens tenham o mesmo tipo.

## C Sharp

```csharp
string[] nomes = {"João", "Maria", "Carlos"};
string[,] pessoas = {
    {"João", "M", "20 anos"},
    {"Maria", "F", "25 anos"}
};
```

C# permite arrays simples e matrizes com duas dimensões usando `[,]`.

## Python

```python
nomes = ["João", "Maria", "Carlos"]
pessoas = [
    ["João", "M", "20 anos"],
    ["Maria", "F", "25 anos"]
]
```

Python usa listas, que são bem parecidas com arrays de JavaScript.

## PHP

```php
$nomes = ["João", "Maria", "Carlos"];
$pessoas = [
    ["João", "M", "20 anos"],
    ["Maria", "F", "25 anos"]
];
```

PHP também usa arrays com colchetes, parecido com JavaScript.
