# Comparando Operadores Lógicos Com Outras Linguagens

Operadores lógicos juntam condições. Os mais comuns são E, OU e NÃO.

## Java

```java
int idade = 18;
boolean documentoValido = true;
boolean podeEntrar = idade >= 18 && documentoValido;
```

Java usa `&&` para E, `||` para OU e `!` para NÃO.

## C Sharp

```csharp
int idade = 18;
bool documentoValido = true;
bool podeEntrar = idade >= 18 && documentoValido;
```

C# usa os mesmos símbolos lógicos de JavaScript.

## Python

```python
idade = 18
documento_valido = True
pode_entrar = idade >= 18 and documento_valido
```

Python usa palavras: `and`, `or` e `not`.

## PHP

```php
$idade = 18;
$documentoValido = true;
$podeEntrar = $idade >= 18 && $documentoValido;
```

PHP aceita `&&`, `||` e `!`, assim como JavaScript.
